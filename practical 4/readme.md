## (iv) Singly Linked List Operations

- **File:** `linkedlist.py`  
- **Concepts Covered:**  
  - Node creation and linking  
  - Singly Linked List structure  
  - Basic operations: `append`, `insert`, `delete`, `search`, and `reverse`  
  - Class and object-based implementation  
  - Traversing linked list using loops  

### Description:
This program demonstrates how to create and manipulate a **Singly Linked List** in Python.  
It defines two classes:  
- **`Node`** — represents each element (node) in the list, containing data and a pointer (`next`) to the next node.  
- **`LinkedList`** — manages the entire list and provides methods to:
  - Add new nodes (`append`)
  - Insert at specific positions (`insert`)
  - Delete nodes (`delete`)
  - Search for a value (`seearch`)
  - Reverse the list order (`reverse`)
  - Display the list (`display`)
